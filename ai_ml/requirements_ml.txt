#   - Compatible with AVX instructions (most modern CPUs)

#   - 8GB RAM minimum, 16GB+ recommended

#   - NVIDIA GPU with Compute Capability 6.0+
#   - RTX 3060 or higher recommended
#   - 8GB+ VRAM recommended
#   - CUDA 11.8+ and cuDNN 8.6+ required

# Storage:
#   - 2GB minimum for package installation
#   - 10GB+ recommended for model training and experiments

# ============================================================================
# Troubleshooting
# ============================================================================

# Issue: TensorFlow import error
# Solution: pip install --upgrade tensorflow

# Issue: GPU not detected
# Solution:
#   1. Verify NVIDIA drivers: nvidia-smi
#   2. Install CUDA: https://developer.nvidia.com/cuda-downloads
#   3. Install cuDNN: https://developer.nvidia.com/cudnn
#   4. Reinstall TensorFlow: pip uninstall tensorflow && pip install tensorflow[and-cuda]

# Issue: Out of memory during training
# Solution:
#   1. Reduce batch size in model configuration
#   2. Use smaller model (reduce LSTM units, etc.)
#   3. Use tensorflow-cpu instead

# Issue: Slow training on Windows
# Solution: Use WSL2 with Linux for better performance
